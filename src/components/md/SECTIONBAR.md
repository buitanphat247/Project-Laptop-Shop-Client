# SECTIONBAR.md - Component SectionBar

## 1. Tổng quan
SectionBar.js là component thanh bar phụ cho section (filter, action...)

## 2. Props
- children: Nội dung bên trong bar

## 3. Chức năng chính

## 4. Ví dụ sử dụng
```js
<SectionBar>Filter here</SectionBar>
```
